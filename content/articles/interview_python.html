General questions

  • What are Python decorators? Do you have any examples in the mind? Have you write your own? How many ways you know to implement decorator? What is Paramerized decorator?

    Ex.: @ordered - decorator that orders function results in ASC way. What if we need to pass direction argument to it (DESC/ASC)?

    Other simple ex: Logging(print), Evaluate execution time, Notify about exceptions, Cache logic (default lru cache decorator), Validation

  • Awareness of basic packages in Python: abc, argparse, re, uuid, logging, itertools, functools, random, os, csv
  • Context Managers. How they are used? Have you write your own?

  • Iterators and Generators in Python. Why do you need them? Have you wrote your own?

    {{}}

    An iterator is an object that contains a countable number of values. Iterator - __iter__() (initialize and return self) and __next__() (next item or Exception)

    Generator in python is a subclass of Iterator. Generators are about 'yield' keyword. Allows you to implement iterations simplier. Generator to optimize memory. Could be defined in inline expression or function. (generator comprehension) Python generator saves the states of the local variables every time ‘yield’ pauses the loop in python. All the local variables before the yield statements are stored.
